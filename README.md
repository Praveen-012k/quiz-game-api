# Quiz Game with API

An interactive True/False quiz game that fetches questions from the Open Trivia Database API. Built with Python and Tkinter.

## Features

- Fetches random trivia questions from external API
- Clean GUI with visual feedback (green for correct, red for wrong)
- Score tracking
- Object-oriented design with separate classes

## Technologies Used

- Python
- Tkinter (GUI)
- Requests (API calls)
- OOP principles

## How to Run

1. Clone this repository
2. Install required packages: `pip install requests`
3. Run: `python main.py`

## Project Structure

- `main.py` - Main application entry point
- `data.py` - API data fetching
- `question_model.py` - Question class
- `quiz_brain.py` - Quiz logic
- `ui.py` - GUI interface
